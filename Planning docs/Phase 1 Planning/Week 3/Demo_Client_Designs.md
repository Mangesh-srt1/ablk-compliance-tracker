# Demo Client Designs Description

## Overview
UI designs for demo client (wallet scans, anomaly detection, reports) with multi-language support.

## Key Flows
- **Wallet Scan**: Input address, scan button, progress bar.
- **Anomaly Detection**: List of flagged txs with reasons.
- **Report Generation**: PDF export with jurisdiction-specific data.

## Features
- Localization toggles (EN/HI/DE).
- Real-time updates.
