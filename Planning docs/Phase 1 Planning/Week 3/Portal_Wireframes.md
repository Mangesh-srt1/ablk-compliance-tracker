# Portal Wireframes Description

## Overview
Wireframes for Ableka Portal using Figma. Includes onboarding, scan initiation, results pages with jurisdiction selection.

## Key Screens
- **Onboarding**: Form for API key generation, jurisdiction dropdown (IN/EU/US).
- **Scan Initiation**: Input entityId, select checks (KYC/AML/Fraud), start button.
- **Results**: Table of scans with status, flags, download report.

## Features
- Multi-language placeholders.
- Responsive design.
