# PRD Draft v1: Ableka Lumina

## Introduction
Ableka Lumina is an AI-driven RegTech SaaS for KYC/AML/fraud detection in fintech, supporting global jurisdictions.

## Core Features
- API-first platform with REST/WebSocket.
- AI agents for compliance checks.
- Portal for client management.
- Multi-tenant, localized.

## UI Flows
- Developer: Onboard → Integrate API → Scan → Report.
- Admin: Login → Review Alerts → Generate Reports.

## Non-Functional
- Performance: <2s response.
- Security: GDPR/DPDP compliant.
- Scalability: 100k scans/day.
